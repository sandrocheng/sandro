1.手机防盗都能做什么？

手机被盗后，防盗功能可以帮您遥控手机做以下事情：

◇ 定位手机：远程定位手机的当前位置，便于您找回被盗手机

◇ 销毁数据：彻底销毁被盗手机中的短信、照片、通讯录等数据，防止个人信息被泄露或盗用，保护隐私安全

◇ 锁定拍照：锁定被盗手机的屏幕，防止小偷使用您的手机。当小偷尝试解锁时，拍下其照片。

◇ 发报警音：让被盗手机发出最大音量的警报声（即使手机在静音模式），方便您寻声查找

◇ 换卡短信通知：手机SIM卡被更换后，会自动发短信到安全手机号
    您可从短信中获取手机的新号码，然后联系对方索回，或发送防盗指令遥控手机

2.如何使用手机防盗？

用任意手机发送防盗指令到被盗手机即可

防盗指令生效后，您会收到短信通知

防盗指令如下：

◇ 定位手机：dingwei#防盗密码

◇ 销毁数据：xiaohui#防盗密码

◇ 锁定拍照：suoding#防盗密码

◇ 发报警音：baojing#防盗密码

以定位手机为例，如您的防盗密码为123456，则定位手机指令为dingwei#123456

特别注意：销毁数据指令必须使用安全手机号发送；其他指令可用任意手机发送

3.安全手机号有什么用？

◇ 接收手机换卡时的短信通知，得到被盗手机的新号码

◇ 重置防盗密码

◇ 发送销毁数据指令

强烈建议您将亲友手机号设置为安全手机号

4.忘记防盗密码怎么办？

用安全手机号向您的手机发送短信指令：chongzhimima
稍等片刻，安全手机号就会收到新密码
